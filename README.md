# UGP_1_IITK_Comp_Stats

Project repository for the reports and programs for my first undergraduate project at IIT Kanpur, supervised by Prof. Dootika Vats. This repository contains analysis code (R), supporting figures, and final report submissions related to applied problems in computational statistics.

## Contents

- `Final Submissions/` — Final report files and submission materials.  
- `Images/` — Figures used in the report and scripts.  
- `Papers/` — Relevant literature and reading materials.  
- `*.R` scripts — Primary R scripts used for analysis and computation.  
- `.gitignore`, `.DS_Store` — Repository housekeeping files.

## Project Summary

This undergraduate project investigates applied problems in computational statistics. The work focuses on statistical modeling, optimization, and data analysis using R, along with reproducible figure generation and report writing.
